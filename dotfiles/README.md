# Dotfiles (work in progress)

The following files belong in the specified locations:

* nobeep.conf -> /etc/modprobe.d/nobeep.conf
* personal.map -> /usr/share/kbd/keymaps/personal.map
* vconsole.conf -> /etc/vconsole.conf
* locale.gen -> /etc/locale.gen
* locale.conf -> /etc/locale.conf
* hostname -> /etc/hostname
* hosts -> /etc/hosts
* crypttab -> /etc/crypttab
* mkinitcpio.conf -> /etc/mkinitcpio.conf
* grub -> /etc/default/grub
* mirrorupgrade.hook -> /etc/pacman.d/hooks/mirrorupgrade.hook
* mdadm.conf -> /etc/mdadm.conf
* sudoers -> /etc/sudoers
* resolv.conf -> /etc/resolv.conf
* network.service -> /etc/systemd/system/network.conf
